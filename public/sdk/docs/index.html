Connecting of Things SDK Documentation

Your guide to interacting with the API / Mock Server

Introduction

The Mock Server SDK provides developers with the tools and resources necessary to interact with the Mock Server. This server allows you to simulate API endpoints, enabling you to develop and test your applications without relying on a live, fully implemented backend.

This SDK simplifies the process of making requests, handling responses, and managing authentication. It also provides a testing environment to experiment with the API.

SDK Overview

The SDK includes the following components, designed to streamline your integration:

  • Documentation Comprehensive guides to the Mock Server API, including endpoints, authentication, and error handling.
  • Examples Practical code examples in PHP and JavaScript demonstrating how to use the SDK.
  • Testbed An interactive environment for testing API calls and viewing responses.

Quick Start Guide

Here's how to get started with the Mock Server SDK:

  1. Set up the Mock Server: Ensure that the Mock Server is installed and running. Refer to the Mock Server installation instructions for details.
  2. Access the SDK: The SDK documentation and tools are located in the public/sdk/ directory of your Mock Server installation.
  3. Explore the Documentation: Start with the API Reference to understand the available endpoints and how to use them.
  4. Learn about Authentication: If the Mock Server requires authentication, see the Authentication section for details on obtaining and using credentials.
  5. View the Examples: Check out the Examples section for code samples in your preferred language.
  6. Use the Testbed: Use the interactive Testbed environment to make API calls and see the responses.
